## 一、预定义字符类

> 表示匹配一类的字符集，由正则表达式的语法提供。如下：

|预预定术语|匹配内容|
|:-- | :-- |
|.|匹配处理新行\n之外的任意字符|
|  |  |
| \s  | 空白字符 |
| \S  |  非空白字符 |
|  |  |
| \w  |  可以构成单词的字符 |
| \W  |  不能构成单词的字符 |
|  |  |
| \d  | 数字 |
| \D  | 非数字 |
|  |  |
| \b | 单词的边界 |
| \B | 不是单词的边界 |
|  |  |
|\t|水平制表符|
|\v|垂直制表符|
|\f|换页符|
|\r|回车符|
|\n|换行符|
|||
|\cA : \cZ|控制符，例如\cM 匹配一个ctrl+m|
|\x0000 : \xFFFF|十六进制的Unicode码|
|\x00 : \xFF|十六进制ASCII码|
